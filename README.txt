
# Angularjs Rating Directive 
Supports Angular1x 

## Steps:
1. download from https://github.com/venkatch668/angularjs-rating-directive.git
Directly can run not required any files to install.


## Features
1. Angular1x
2. Easily customizable(ex:Start, Bars) 


## Values Need Pass
1. pass the value of rating into "ng-model"
2. to get return value use "on-rating-selected" rateFunction
3. can set max value  using "max" attribute   
  <skill-rating ng-model="rating1" max="5" on-rating-selected="rateFunction(rating)"></skill-rating>
